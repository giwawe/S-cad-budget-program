# 真实模板报价复核动作决策表

来源：`scratch/cad-import-10-real-template-priced-command/quote-priced-review.json`。

当前正式报价包的复核行动共 6 类：`high` 3 类、`medium` 3 类。所有行动都来自默认推断，当前没有 `自动生成-异常提示` 和 `按模板生成` 行。

## 当前可接受默认

- 厨房推拉门缺高按 2.2m，已符合当前业务口径，但仍保留复核提示。
- 未识别阳台/露台推拉门时，阳台推拉门及双包套按 0 生成；本次没有要求自动纳入。
- `砖墙门窗洞过梁`、`背景墙`、`入户门`、`玻璃淋浴房`、`蹲坑` 这类按需项目未标识时按 0 自动生成，不占用默认推断状态。

## 下一轮处理建议

优先顺序建议为：窗高、新砌墙高度/厚度、外墙修补范围，然后再处理管道/包管、全屋定制和门洞高度。窗高影响面最大，会联动墙面净面积、墙砖、封门窗和美缝。
